**微观人群模拟方法--基于RVO库&OpenGL库的实现**

**项目中源代码分为2D和3D实现两部分**

​		Room.cpp ：模拟多个房间内多个人群

​		CrossRoads.cpp：模拟十字路口中多个人群

​		Pass.cpp：模拟通道内人多个人群
